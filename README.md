# Magnetic Navigation & Collection Simulation Framework

A modular simulation framework to evaluate magnetic anomaly data collection strategies and magnetic navigation (MagNav) performance in GPS-denied environments.

## Features
- **World Generation**: Simulate magnetic fields using synthetic noise or load real raster data.
- **Aircraft Simulation**: Kinematic model for fixed-wing aircraft.
- **Sensor Modeling**: Magnetometer and Altimeter simulation with configurable noise characteristics.
- **Navigation Algorithms**: Particle Filter (PF) and Dead Reckoning integration.
- **Mission Planning**: Survey pattern generation (e.g., Lawnmower).

## Setup
```bash
pip install -r requirements.txt
```
